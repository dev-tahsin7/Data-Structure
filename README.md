## 📘 Folder Details

### `Modules/`

This is the main folder of the repository. Each module contains the topics taught in the course.

Inside every module, the files are organized topic-wise and class-wise so that I can easily track my learning progress.

Each topic may include:

* **Class Code:** Code written during class or while learning the topic.
* **Practice Problems:** Problems solved to strengthen the concept.
* **Resources:** Notes, references, links, or explanations related to the topic.
